# Site Simples

Este projeto foi desenvolvido como parte de um curso da **Fundação Bradesco**.

O objetivo é praticar os conceitos básicos de desenvolvimento web, utilizando tecnologias fundamentais para a criação de um site simples.

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript (se tiver, senão pode remover)

## Observação
Projeto com fins educacionais, voltado para aprendizado e prática.
